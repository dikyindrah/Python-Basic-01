Tipe Data Berurut

Tipe Data Berurut adalah jenis data yang dapat berisi satu atau beberapa nilai 
didalamnya, nilaiâ€“nilai ini sering juga disebut item, elemen, atau anggota. Disebut
tipe data berurut karena setiap nilai yang ada didalam tipe data ini mememiliki 
indeks yang tersusun secara urut dimulai dari 0.

Bahasa pemrograman python memiliki dua tipe data berurut yang dapat digunakan yaitu 
List dan Tuple.


List

List adalah tipe data yang berisi satu atau beberapa nilai di dalamnya. List bersifat 
mutable yang artinya data di dalamnya bisa diubah. Nilai pada list bisa terdiri dari tipe
data lain seperti tipe data numerik, string, boolean bahkan campuran ketiganya.

List dibuat dengan menempatkan semua item di dalam tanda kurung siku [ ], dan untuk
stiap item yang berbeda dipisahkan oleh tanda koma. 

Contoh:

# Struktur penulisan variabel dengan tipe data list
nama_variabel = [item, item, item]

# List kosong
nilai = [ ]

# List berisi integer
nilai_integer = [1,2,3,4,5]

# List berisi tipe campuran
nilai_campuran = [1, 3.5, "Hello"]


Item pada List juga bisa diisi dengan list lain. Ini disebut dengan List bersarang.

Contoh:

# List bersarang 
list_bersarang = ["hello", [2,4,6], ['a','b']]


Kita bisa mengakses item pada list menggunakan indeks yang ada, dengan format namalist[indeks].
Untuk mengakses item list secara urut dari awal kita dapat memulainya dari indeks ke 0 dan untuk 
untuk mengakses item list secara urut dari akhir kita dapat memulainya dengan indeks negatif
mulai dari -1.

List yang memiliki 5 item akan memiliki 4 indeks, dimulai dari 0 s/d 4. Mencoba mengakses item 
pada list di luar itu akan menyebabkan error IndexError.

Contoh:

# List berisi integer
nilai_integer = [34, 23, 98]

# Mengakses dan menampilkan indeks ke 2
print(nilai_integer[2])

# Mengakses dan menampilkan indeks ke -1
print(nilai_integer[-1])

# List bersarang
list_bersarang = ["hello", [2,4,6], ['a','b']]

# Mengakses dan menampilkan indeks ke 1,1
print(list_bersarang[1,1])


Tuple

Tuple Sama seperti list, bedanya adalah jika list dimulai dan diakhiri dengan kurung siku,
maka tuple menggunakan kurung biasa. Perbedaan selanjutnya terdapat pada sifatnya, tuple 
bersifat immutable artinya data yang didalamnya tidak bisa diubah. 

Nilai pada tuple juga bisa terdiri dari tipe data lain seperti tipe data numerik, string, 
boolean bahkan campuran ketiganya.

Tuple dibuat dengan menempatkan semua item di dalam tanda kurung ( ), dan untuk
stiap item yang berbeda dipisahkan oleh tanda koma. 

Contoh:

# Struktur penulisan variabel dengan tipe data tuple
nama_variabel = (item, item, item)

# Tuple kosong
nilai = ()

# Tuple berisi integer
nilai_integer = (1,2,3,4,5)

# Tuple berisi tipe campuran
nilai_campuran = (1, 3.5, "Hello")


Item pada tuple juga bisa diisi dengan tuple atau list. Ini disebut dengan tuple bersarang.

Contoh:

tuple_bersarang = ("hello", [1, 2, 3], (4, 5, 6)) 


Seperti halnya list, kita bisa mengakses item pada tuple lewat indeksnya menggunakan format 
namatuple[indeks]. Untuk mengakses item tuple secara urut dari awal kita dapat memulainya dari 
indeks ke 0 dan untuk untuk mengakses item tuple secara urut dari akhir kita dapat memulainya 
dengan indeks negatif mulai dari -1.

Contoh:

# Tuple berisi integer
nilai_integer = (34, 23, 98)

# Mengakses dan menampilkan indeks ke 2
print(nilai_integer[2])

# Mengakses dan menampilkan indeks ke -1
print(nilai_integer[-1])

# Tuple bersarang
tuple_bersarang = ("hello", [1, 2, 3], (4, 5, 6)) 

# Mengakses dan menampilkan indeks ke 1,2
print(Tuple_bersarang[1][2])


Setelah tuple dibuat, maka item pada tuple tidak bisa lagi diubah atau dihapus. Akan tetapi, 
bila item tuple-nya adalah tuple bersarang dengan item seperti list, maka item pada list tersebut
dapat diubah. 

Contoh:

nilai_integer = (34, 23, 98)

# Saat kita memaksa mengubah item pada tuple nilai_integer di indeks ke 1 
# Akan muncul error: # TypeError: 'tuple' object does not support item assignment 
# Artinya kita tidak bisa mengganti nilai yang sudah ada pada tuple dengan nilai yang baru
nilai_integer[1] = 12


tuple_bersarang = ("hello", [1, 2, 3], (4, 5, 6)) 

# Tapi item pada list di dalam tuple_bersarang bisa diubah 
tuple_bersarang[1][0] = 8
