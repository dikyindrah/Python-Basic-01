Catatan:

Return

Return adalah pernyataan khusus yang digunakan pada fungsi (Function) untuk 
mengirim hasil dari eksekusi pernyataan suatu fungsi kembali ke bagian 
dimana fungsi itu dipanggil. Return bisa kita isi dengan ekspresi. Jika 
ekspresi dikosongkan, maka fungsi akan mengembalikan objek None.

Hasil yang dikembalikan bisa berupa nilai atau objek apa pun. Misal-nya 
seperti nilai-nilai dengan tipe date numerik (integer, float, dan complex), 
teks (String), boolean, urutan (list, tuple, set, dan dictionary), kelas, 
fungsi, modul atau paket, dll.

Contoh:

# Return dengan ekspresi
def penjumlahan(x, y):
    return x + y

# Return tidak dengan ekspresi
def perkalian(x, y):
    return

hasil_penjumlahan = penjumlahan(5, 5)
print(hasil_penjumlahan)

hasil_perkalian = perkalian(2, 5)
print(hasil_perkalian)


Output:

10
None
