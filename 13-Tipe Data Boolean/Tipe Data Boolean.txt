Tipe Data Boolean

Tipe Data Boolean adalah jenis data dalam bentuk logika yang hanya 
dapat menyatakan nilai True dan False. Untuk mendeklarasikan 
variabel dengan tipe data boolean kita dapat langsung mengisi 
variabel tersebut dengan salah satu dari nilai True dan False.

Contoh:

# Struktur penulisan variabel dengan tipe data boolean
nama_variabel = nilai boolean

# Variabel dengan tipe data boolean bernilai true
saya_lapar = True

# Variabel dengan tipe data boolean bernilai false
saya_sedih = False

print(saya_lapar)
print(saya_sedih)


Output:

True
False


Penggunaan tipe data boolean akan lebih terlihat saat kita membuat 
program dengan struktur kontrol seperti pemilihan (IF dan Elif) 
atau perulangan (For dan While).
