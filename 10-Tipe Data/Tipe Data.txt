Tipe Data 

Dalam pemrograman komputer, tipe data adalah klasifikasi yang menentukan jenis data yang
akan disimpan dan dimiliki sebuah variabel.

Pada bahasa pemrograman python saat kita membuat variabel, kita tidak perlu mendeklarasikan
tipe data apa yang akan digunakan pada variabel tersebut seperti bahasa pemroraman lain. 
Ini karean bahasa pemrograman python memiliki sistem tipe dinamis (Dynamic Typing) yang 
membuat jenis data pada variabel akan otomatis diketahui dari data apa yang kita berikan.

Tipe Data yang dapat digunakan pada bahasa pemrograman python:

1.  Tipe Data Text
    -   String

2.  Tipe Data Numeric
    -   Integer
    -   Float
    -   Complex

3.  Tipe Data Boolean

4.  Tipe Data Sequence
    -   List
    -   Tuple
    -   Set
    -   Dictionary
