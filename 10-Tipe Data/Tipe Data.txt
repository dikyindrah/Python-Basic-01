Tipe Data 

Memori komputer dapat menyimpan data dengan berbagai tipe. Namun bagaimana cara kita 
memberitahu komputer untuk mengklasifikasikan data sesuai tipe yang kita inginkan?

Hal itu dapat diselesaikan dengan penggunaan tipe data. Tipe data akan mendefinisikan 
atau menjelaskan jenis data apa yang kita masukan ke dalam variabel menggunakan pemahaman 
komputer.

Pada bahasa pemrograman python saat kita membuat variabel, kita tidak perlu mendeklarasikan
tipe data apa yang akan digunakan pada variabel tersebut seperti bahasa pemroraman lain. 
Ini karena bahasa pemrograman python memiliki sistem tipe dinamis (Dynamic Typing) yang 
membuat jenis data pada variabel akan otomatis diketahui dari data apa yang kita berikan.

Tipe Data yang dapat digunakan pada bahasa pemrograman python:

1.  Tipe Data Teks
    -   String

2.  Tipe Data Numerik
    -   Integer
    -   Float
    -   Complex

3.  Tipe Data Boolean

4.  Tipe Data Berurut
    -   List
    -   Tuple

5.  Tipe Data Tidak Berurut
    -   Set
    -   Dictionary
