Windows

1.  Kunjungi sistus resmi python https://www.python.org/downloads/.
2   Download python versi terbaru dan tunggu proses download hingga selesai.
3.  Setelah proses download selesai, selanjutnya adalah menginstal python.
    Klik dua kali pada file installer python untuk membuka dan memulai proses
    insalasi.
4.  Pada awal insalasi akan muncul menu "Instal Python 3.9.0 (32-bit)", pada
    menu ini terdapat dua check box, yang terdiri dari:

    [ ] Check box pertama   - Instal launcher for all user (Recomended)
    [ ] Check box kedua     - Add python 3.9 to PATH

    jangan lupa untuk menceklis keduanya.
5.  Jika sudah klik "Instal Now" untuk memulai proses pemasangan python ke
    komputer kita.
6.  Tunggu proses pada menu "Setup Progres" hingga selesai.
7.  Jika proses telah selesai akan tampil menu "Setup was successful", pada
    menu ini jangan lupa untuk klik "Disable path length limit"
8.  Jika sudah klik close.
9.  Selanjutnya adalah memeriksa apakah python sudah benar-benar berhasil
    terpasang pada komputer kita, caranya dengan mengetikan perintah berikut
    pada CLI:

    python --version : Untuk melihat versi python yang telah terpasang pada komputer.
    python           : Untuk memanggil atau menjalankan mode interaktif python.
