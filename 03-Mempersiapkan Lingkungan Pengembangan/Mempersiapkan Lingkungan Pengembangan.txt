Source-code Editor

Source-code Editor adalah program editor teks yang dirancang khusus untuk mengedit kode sumber program 
komputer. Ini mungkin aplikasi mandiri atau mungkin dibangun ke dalam lingkungan pengembangan terintegrasi (IDE) 
atau browser web. Editor kode sumber adalah alat pemrograman dasar, karena tugas dasar programmer adalah 
menulis dan mengedit kode sumber.

Source-code Editor memiliki fitur yang dirancang khusus untuk menyederhanakan dan mempercepat pengetikan 
kode sumber, seperti fungsi syntax highlighting, indentation, autocomplete, dan pencocokan tanda kurung. 
Editor ini juga menyediakan cara mudah untuk menjalankan kompiler, interpreter, debugger, atau program lain 
yang relevan untuk proses pengembangan perangkat lunak. Jadi, meskipun banyak editor teks seperti Notepad 
dapat digunakan untuk mengedit kode sumber, jika mereka tidak menyempurnakan, mengotomatiskan, atau memudahkan 
pengeditan kode, mereka bukanlah Source-code Editor.

Contoh Source-code Editor : Visual Studio Code, Atom, Sublime Text dan Notepad++.


Integrated Development Environment (IDE)

Integrated Development Environment adalah program komputer yang memiliki beberapa fasilitas yang 
diperlukan dalam pembangunan perangkat lunak. Tujuan dari IDE adalah untuk menyediakan semua utilitas 
yang diperlukan dalam membangun perangkat lunak. 

Sebuah IDE, atau secara bebas dapat diterjemahkan sebagai Lingkungan Pengembangan Terpadu, setidaknya 
memiliki fasilitas:

Editor      -   Fasilitas untuk menuliskan kode sumber dari perangkat lunak.
Compiler    -   Fasilitas untuk mengecek sintaks dari kode sumber kemudian mengubah dalam bentuk 
                binari yang sesuai dengan bahasa mesin.
Linker      -   Fasilitas untuk menyatukan data binari yang beberapa kode sumber yang dihasilkan 
                compiler sehingga data-data binari tersebut menjadi satu kesatuan dan menjadi suatu program 
                komputer yang siap dieksekusi.
Debuger     -   Fasilitas untuk mengetes jalannya program, untuk mencari bug/kesalahan yang terdapat 
                dalam program.

Sampai tahap tertentu IDE modern dapat membantu memberikan saran yang mempercepat penulisan. Pada saat 
penulisan kode, IDE juga dapat menunjukan bagian-bagian yang jelas mengandung kesalahan atau keraguan. 

Contoh IDE untuk python : PyCharm, Spyder, Eclipse (Menggunakan plugin Pydev) dan Eric.
