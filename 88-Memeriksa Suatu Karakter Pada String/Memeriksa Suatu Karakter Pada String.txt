Catatan:

Memeriksa Suatu Karakter Pada String

Kita dapat memeriksa apakah suatu karakter terdapat 
pada sebuah string menggunakan keyword in dan not in.

Contoh:

teks = 'python'
print('teks =',teks)

# Memeriksa apakah ada karakter x pada string
print('p' in teks)
print('h' in teks)
print('k' in teks)

# Memeriksa apakah tidak ada karakter x pada string
print('k' not in teks)
print('y' not in teks)


Output:

teks = python

True
True
False

True
False
