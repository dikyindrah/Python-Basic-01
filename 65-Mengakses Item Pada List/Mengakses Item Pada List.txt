Mengakses Item Pada List

Setiap item pada list memiliki indeks. Indeks ini bisa kita
akses untuk menggunakan item yang ada pada list.

Indeks setiap item pada list selalu dimulai dari angka 0.

  list_string = ['aku', 'sedang', 'belajar', 'python']
  Indeks           0       1          2          3

list_string memiliki 4 item dengan indeks berakhir pada 
angka 3 atau sederhananya list_string memiliki panjang 4
dan indeks 3 dimulai dari 0.

Untuk menggunakan item yang ada pada list_string kita hanya
perlu mengakses indeks yang ada.

Contoh:

list_string = ['aku', 'sedang', 'belajar', 'python']

print(list_string[0])
print(list_string[1])
print(list_string[2])
print(list_string[3])


Output:

aku
sedang
belajar
python


Agar lebih mudah kita bisa memanfaatkan perulangan untuk
mengakses setiap indeks yang ada.

Contoh:

# Akses list menggunakan for
for item in list_string:
    print(item)

# Akses list menggunakan while
i = 0
l = len(list_string)
while i < l:
    print(i, list_string[i])
    i = i + 1

Output:

aku
sedang
belajar
python

aku
sedang
belajar
python
